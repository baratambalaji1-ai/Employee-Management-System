# Employee Management System - SQL Project

## 📌 Project Overview
This project is designed to manage employee data using SQL and perform analysis on HR datasets.

## 🛠️ Tools Used
- MySQL
- SQL Queries
- PowerPoint (for presentation)

## 📂 Project Files
- SQL Script: Database creation, tables, queries
- PPT: Project explanation and insights

## 📊 Key Features
- Employee data management
- Department & salary analysis
- Leave tracking
- Payroll analysis

## 🔍 Key Insights
- Total Employees: 60
- Finance & IT have highest workforce
- Legal department has highest average salary
- Payroll analysis done for April 2024

## 📈 Concepts Used
- Joins
- Group By
- Aggregate Functions
- Window Functions (RANK)
